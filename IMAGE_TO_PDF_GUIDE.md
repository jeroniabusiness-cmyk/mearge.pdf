# Image to PDF - User Guide

## 🖼️ How to Convert Images to PDF

### Quick Start

1. **Start conversion**
   ```
   Send: /img2pdf
   ```

2. **Send your images**
   - Send images one by one
   - Supports: JPG, PNG, WEBP, BMP, TIFF
   - Can send up to 20 images

3. **Create PDF**
   - Click "Create PDF" button
   - Download your PDF!

## ✨ Features

### Image Management

- **📤 Upload** - Send images via Telegram
- **📋 View List** - See all uploaded images
- **🔄 Rotate** - Rotate any image (90°, 180°, 270°)
- **🗑️ Remove** - Delete unwanted images
- **↕️ Order** - Images are added in send order

### PDF Options

- **📏 Page Size** - Choose from:
  - A4 (210x297mm) - Default
  - Letter (8.5x11")
  - Legal (8.5x14")
  - A3 (297x420mm)
  - A5 (148x210mm)

- **📐 Fit to Page** - Auto-resize images to fit page

### Image Processing

- **Auto RGB Conversion** - All images converted to RGB
- **Orientation Detection** - Auto-detect portrait/landscape
- **Quality Preservation** - High-quality output
- **EXIF Handling** - Proper orientation handling

## 📱 Step-by-Step Guide

### Basic Conversion

```
1. /img2pdf
   ✅ Conversion started

2. Send: photo1.jpg
   ✅ Image 1 added

3. Send: photo2.jpg
   ✅ Image 2 added

4. Click: "Create PDF"
   ⏳ Processing...
   ✅ PDF ready!
```

### With Rotation

```
1. Upload images
2. Click: "View Images"
3. Click: "Rotate Image"
4. Select image to rotate
5. Choose: 90°, 180°, or 270°
   ✅ Image rotated
6. Click: "Create PDF"
```

### Custom Settings

```
1. Upload images
2. Click: "Options"
3. Click: "Change Page Size"
4. Select: Letter
   ✅ Page size set
5. Toggle: "Fit to Page"
   ✅ Fit enabled
6. Click: "Create PDF"
```

## 🔧 Advanced Features

### Rotation

**When to rotate:**
- Wrong orientation
- Landscape/portrait mix
- Scanned pages sideways

**How to rotate:**
```
View Images → Rotate Image → Select → Choose angle
```

**Angles:**
- 90° - Clockwise quarter turn
- 180° - Half turn (upside down)
- 270° - Counter-clockwise (or 3x 90°)

### Page Sizes

**Choosing page size:**

- **A4** - Standard documents (Europe)
- **Letter** - Standard documents (US)
- **Legal** - Legal documents
- **A3** - Large documents/posters
- **A5** - Small booklets

**Fit to Page:**
- ✅ ON - Images resized to fit
- ❌ OFF - Original image size used

### Image Order

**Images are in send order:**
```
First sent = First page
Last sent = Last page
```

**To reorder:**
1. Remove images
2. Re-send in correct order

Or:
1. Send all images
2. Remove unwanted
3. Re-send specific images

## 💡 Tips & Tricks

### Best Practices

✅ **DO:**
- Use high-quality images
- Keep similar orientations together
- Check image list before creating
- Use fit-to-page for consistency

❌ **DON'T:**
- Mix very different sizes
- Send corrupted images
- Exceed file size limits
- Send unsupported formats

### Quality Tips

**For best results:**
- Use original photos (not screenshots)
- Avoid heavily compressed images
- Keep images under 50MB
- Use consistent DPI when scanning

### Efficiency

**Save time:**
- Send multiple images quickly
- Use default settings if suitable
- Create PDF immediately if no edits needed

## 🐛 Troubleshooting

### "Invalid image"
**Cause:** Corrupted or unsupported file
**Solution:** Use JPG, PNG, WEBP, BMP, or TIFF

### "File too large"
**Cause:** Image exceeds 50MB
**Solution:** Compress image before sending

### "Error creating PDF"
**Cause:** Processing issue
**Solution:**
1. Try fewer images
2. Check image formats
3. Use /cancel and restart

### Images in wrong order
**Cause:** Sent in wrong order
**Solution:**
1. Remove incorrect images
2. Re-send in correct order

### PDF too large
**Cause:** Many high-res images
**Solution:**
- Compress images before sending
- Use smaller page size
- Enable fit-to-page
